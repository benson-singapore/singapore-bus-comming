# 快速开始指南

## 如何找到公交站点代码？

公交站点代码是一个5位数字，用于唯一标识一个公交站。

### 方法 1: 使用 ArriveLah 网站

1. 访问 [ArriveLah2](https://arrivelah2.busrouter.sg/)
2. 在地图上找到你要监控的公交站
3. 点击站点，URL 中的 `?id=` 后面的数字就是站点代码
   - 例如: `https://arrivelah2.busrouter.sg/?id=67661`
   - 站点代码是: `67661`

### 方法 2: 查看公交站牌

部分公交站牌上会标注站点代码。

## 常见问题

### Q: 为什么有些公交显示"预计时间"而不是"实时位置"？

A: 这取决于公交车是否安装了GPS追踪设备。显示 📍 表示有实时GPS位置，显示 ⏱️ 表示是基于时刻表的预计时间。

### Q: 数据多久更新一次？

A: 系统每10秒自动从API获取最新数据，同时本地倒计时每秒更新。

### Q: 数据保存在哪里？

A: 所有配置都保存在浏览器的 LocalStorage 中，不会上传到任何服务器。清除浏览器数据会导致配置丢失。

### Q: 支持哪些地区的公交？

A: 目前仅支持新加坡的公交系统（使用 ArriveLah2 API）。

### Q: 可以离线使用吗？

A: 不可以，需要网络连接来获取实时公交数据。

## 状态说明

系统会根据到站时间自动显示不同状态：

- 🔴 **即将到站** (红色脉冲): 60秒内到达
- 🟠 **快到了** (橙色): 1-3分钟内到达
- 🔵 **在路上** (蓝色): 3分钟以上

## 技巧提示

1. **常用站点**: 将你每天乘车的站点都添加进去，一目了然
2. **筛选线路**: 只添加你常坐的线路，避免信息过载
3. **多设备同步**: 在不同设备上手动配置相同的站点
4. **备份配置**: 可以截图保存你的配置，以便重新设置

## 需要帮助？

如果遇到问题，请检查：

1. 站点代码是否正确（5位数字）
2. 网络连接是否正常
3. 浏览器是否支持 LocalStorage
4. 公交线路号是否在该站点运营
